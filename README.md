# 🌦 Weather App
A simple weather app built with HTML, CSS, and JavaScript.

## 🚀 Features
- Get real-time weather data by city name
- Uses OpenWeatherMap API

## 📦 Technologies
- HTML, CSS, JS
- OpenWeatherMap API

## 🔧 How to Run
1. Clone this repo
2. Open weather.html in your browser
